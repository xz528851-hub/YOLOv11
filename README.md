## 1. 项目摘要

本项目针对薄膜电阻激光调阻工艺中，刻痕结构呈现出的细长、低对比度、边缘模糊等特征，以及工业界对微米级精确测量的迫切需求，设计并实现了一套完整的工业视觉检测系统。

系统创新性地采用了“两级检测（YOLOv11）+ 图像增强（Real-ESRGAN）+ 几何拟合测量”的技术路线。通过从“粗定位”到“精测量”的级联架构，不仅解决了传统算法难以同时兼顾多目标检测效率与微小结构测量精度的矛盾，更实现了从单纯的“定性识别”向高精度的“定量测量”跨越，显著提升了电阻产品的质量管控水平。

## 3. 核心技术栈

- **深度学习框架**：PyTorch, Ultralytics (YOLOv11)
- **图像增强算法**：Real-ESRGAN (超分辨率重建)

- **交互界面框架**：Gradio (Web UI)
- **开发环境**：Python 3.8+

## 4. 系统总体架构

本系统构建了从物理图像采集到数字化质量判定的完整闭环。整体流程分为 8 个核心阶段，涵盖了数据预处理、模型推理、后处理及物理量计算：

1.  **比例尺标定**：建立像素坐标系与物理尺寸（mm）的映射关系。
2.  **电阻区域检测**：在多电阻原图中定位单个电阻区域（一级 YOLOv11 模型）。
3.  **单电阻裁剪**：对检测到的区域进行外扩裁剪，保留必要的上下文信息。
4.  **图像超分增强**：利用 Real-ESRGAN 模型提升刻痕边缘的清晰度与信噪比。
5.  **刻痕线条检测**：在增强后的图像中精细化识别刻痕位置（二级 YOLOv11 模型）。
6.  **候选框后处理**：执行去重、边界修整及交叠消解，优化检测结果。
7.  **几何参数测量**：基于带状投影拟合算法，计算刻痕的长度、宽度、间距及平行度。
8.  **质量判定与展示**：依据规格标准输出 Pass/Fail 结果，并生成可视化报表。

## 5. 关键技术详解

### 5.1 图像增强与预处理

针对工业现场光照不均以及微小刻痕特征不明显的问题，项目引入了 Real-ESRGAN 超分辨率重建算法。该模块在检测前对感兴趣区域进行增强，显著提升了微小线条的信噪比，解决了模糊边缘难以精确框选的难题，为后续的高精度测量奠定了数据基础。

![图2：Real-ESRGAN 图像增强效果对比](photo/2.png)
*图 2：采用 Real-ESRGAN 对单电阻区域图像进行超分辨增强的前后效果对比*

### 5.2 两级 YOLOv11 检测架构

为了提高检测效率并降低背景噪声干扰，系统采用了“由粗到精”的两级检测策略：

- **第一级检测（ROI 定位）**：使用 YOLOv11 模型在包含多个电阻的原始大图中定位每个电阻的边界框，实现背景剔除。
- **第二级检测（细节识别）**：将裁剪出的单电阻图像输入到专门针对线条训练的 YOLOv11 模型中，实现对微米级刻痕的高精度识别。

![图3：两级检测流程示意图](photo/3.png)
*图 3：从原图定位电阻到裁剪区域识别刻痕的两级检测流程*

### 5.3 基于带状投影拟合的测量算法

这是本项目的核心创新点。不同于传统目标检测仅输出矩形框，本项目实现了亚像素级的几何测量：

- **中心带提取**：统计图像行的暗响应分布，确定刻痕的中心区域。
- **中心线拟合**：逐列搜索灰度最小值点，利用直线拟合算法计算刻痕中心线。
- **参数计算**：
    - **长度**：计算中心线两端点的欧氏距离并乘以比例尺。
    - **宽度**：计算中心带边界的间距并乘以比例尺。
    - **平行度**：计算相邻线条中心线方向角的差值。

## 6. 软件界面与交互

系统基于 Gradio 开发了友好的 Web 交互界面，支持用户进行全流程操作：

- **可视化检测**：支持上传原始图像，系统自动在原图上叠加显示刻痕检测框、尺寸文本及中心线。
- **数据导出**：支持导出包含 ROI 汇总表（合格/不合格）及刻痕明细表（长宽/间距/角度）的 CSV 文件，便于后续数据分析。
