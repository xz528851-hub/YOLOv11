# 薄膜电阻激光调阻刻痕智能检测系统
基于 YOLOv11 + Real-ESRGAN 的国产激光调阻设备微米级刻痕自动化检测方案

## 项目简介
针对国产激光调阻设备刻痕检测精度不足、微小刻痕识别适配性差、产线检测效率低等痛点，本项目提出并实现**「显微采集—算法增强—智能检测」**全栈式解决方案。构建「由粗到精」两级智能检测策略，结合深度学习超分增强与目标检测技术，实现薄膜电阻刻痕的高精度识别、测量与质量判定，有效提升工业产线检测良率与自动化水平。

## 技术栈
- 目标检测：YOLOv11
- 图像增强：Real-ESRGAN 超分辨率算法
- 部署框架：Python + 网页交互系统

## 核心功能
1. 电子显微镜显微图像自动化采集与预处理
2. 多电阻目标实时检测、横向排序与区域裁剪
3. 低分辨率刻痕图像深度学习超分增强
4. 微小刻痕智能识别、几何参数测量与合格判定

## 项目流程
1. 图像采集 → 2. 电阻区域定位裁剪 → 3. 超分图像增强 → 4. 刻痕智能检测 → 5. 结果输出

## 创新点
1. 提出「显微采集—算法增强—智能检测」全栈式方案，适配产线实际检测需求
2. 构建「由粗到精」两级检测策略，降低复杂背景干扰
3. 融合 Real-ESRGAN 超分技术，提升低清显微图像刻痕边缘清晰度
4. 基于 YOLOv11 实现多电阻自动排序与标准化裁剪

## 项目成果
1. 构建薄膜电阻刻痕检测全流程工业级方案
2. 实现电阻定位、图像增强、刻痕检测端到端自动化处理
3. 完成 450+ 张电阻显微图像数据集构建与模型训练
4. 申请相关发明专利 1 项，具备工业落地应用价值
